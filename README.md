# godot-action-adventure-game

Este es un juego Zelda-Like de accion-aventura sencillo desarrollado usando [Godot Engine](https://godotengine.org/).

## Controles

- Moverse: WASD
- Correr: SHIFT
- Inventario: TAB
- Atacar: SPACE or Left Click
- Interactuar: E
